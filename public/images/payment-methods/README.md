# Payment Method Logos

This directory contains logos for payment methods used in the checkout flow.

## Required Files

Place the following logo files in this directory:

1. **mada-logo.png** - MADA payment network logo
   - Recommended size: 200x80px (or similar aspect ratio)
   - Format: PNG with transparent background
   - Source: [MADA Official Website](https://mada.com.sa) or [MADA Brand Guidelines](https://gate2play.docs.oppwa.com/reference/brands-reference)

2. **visa-logo.png** - Visa payment card logo
   - Recommended size: 200x80px (or similar aspect ratio)
   - Format: PNG with transparent background
   - Source: [Visa Brand Center](https://brandcenter.visa.com/) or use publicly available logo

3. **mastercard-logo.png** - Mastercard payment card logo
   - Recommended size: 200x80px (or similar aspect ratio)
   - Format: PNG with transparent background
   - Source: [Mastercard Brand Center](https://brand.mastercard.com/) or use publicly available logo

## Image Guidelines

- **Format**: PNG with transparent background (preferred) or SVG
- **Size**: Minimum 200px width, maintain aspect ratio
- **Style**: Official brand logos with proper colors
- **Background**: Transparent (PNG) or white background acceptable

## Fallback Behavior

If logos are not available, the component will display the first 2 letters of the payment method name as a fallback.

## Quick Setup

You can download official logos from:
- **MADA**: Check HyperPay documentation or MADA official resources
- **Visa**: Available from Visa Brand Center (requires account)
- **Mastercard**: Available from Mastercard Brand Center (requires account)

For quick testing, you can use publicly available logos from CDN services, but ensure you have proper licensing for production use.
